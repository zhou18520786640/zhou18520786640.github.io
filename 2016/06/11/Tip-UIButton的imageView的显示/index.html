Tip:UIButton的imageView的显示

发表于 2016-06-11

场景

当我们在为视图控制器添加leftBarItem或者rightBarItem的时候，视觉工程师给我们的图的位置总不是苹果设计好的位置。而leftBarItem的customview又不是自己能决定的。这时候我们可以利用UIButton的一些属性来达到想要的位置和效果。

setImage:...与setBackgroundImage:...

使用setImage:会保持image大小不变，让内部的ImageView和image一样大，并将ImageView居中显示在button上
使用setBackgroundImage也会让image大小和button一样大，并且让imageview也和image，button一样。
总结来说。一个不会拉伸图像。一个会拉伸图像。但无论拉伸与否，都会让imageView的尺寸和image一样。
如图1，图2

imageEdgeInsets调整位置

UIEdgeInsets 是一个结构体,描述了一个视图与父视图的直接的间距。

typedef struct UIEdgeInsets {
    CGFloat top, left, bottom, right;  // specify amount to inset (positive) for each of the edges. values can be negative to 'outset'
} UIEdgeInsets;

正数表示离父视图内部的距离。负数表示离父视图外部的距离。top,bottom都是垂直维度的变量。left,right是水平维度的变量。我们只需要改变垂直维度的其中一个变量即可影响垂直位置。同理改变left就可以改变水平维度（right = 0 ） 。当我们使用setImage:设置imageView后，发现imageView水平位置不满足要求,可以这样设置

button.imageEdgeInsets = UIEdgeInsetsMake(28,0,0,0);

28将覆盖原来相对button垂直居中的布局。直接将让imageView垂直方向下与button间隔28.因为我们让top值为28.因为不再需要设置bottom.(bottom = 0)依然不变。注意这个时候不会对button进行拉伸。同时也不会改变水平位置。依然水平居中。就如图3。

如果要改变水平方向。可以设置

button.imageEdgeInsets = UIEdgeInsetsMake(0,28,0,0);

完整demo

#import "ViewController.h"

@interface ViewController ()

@end

@implementation ViewController

- (void)viewDidLoad {
    [super viewDidLoad];

    {   // 不自动铺满，image维持原来的位置,并且居中
        UIButton *button = [UIButton buttonWithType:UIButtonTypeCustom];
        button.frame = CGRectMake(20, 20, 44, 44);
        [button setImage:[UIImage imageNamed:@"add"] forState:UIControlStateNormal];
        button.backgroundColor = [UIColor redColor];
        [self.view addSubview:button];
    }

    {   // 自动铺满，不管uiimage 本身的大小
        UIButton *button = [UIButton buttonWithType:UIButtonTypeCustom];
        button.frame = CGRectMake(20, 70, 44, 44);
        [button setBackgroundImage:[UIImage imageNamed:@"add"] forState:UIControlStateNormal];
        button.backgroundColor = [UIColor redColor];
        [self.view addSubview:button];
    }


//    UIKIT_STATIC_INLINE UIEdgeInsets UIEdgeInsetsMake(CGFloat top, CGFloat left, CGFloat bottom, CGFloat right) {

    {   // image维持原来的尺寸.
        UIButton *button = [UIButton buttonWithType:UIButtonTypeCustom];
        button.frame = CGRectMake(20, 130, 44, 44);
        [button setImage:[UIImage imageNamed:@"add"] forState:UIControlStateNormal];
        button.backgroundColor = [UIColor redColor];

        // top 让他不再遵从垂直方向的居中。而是以button顶部内边距28
        // left,right等于0 表示依然遵从水平方向居中
        button.imageEdgeInsets = UIEdgeInsetsMake(28,0,0,0);
        [self.view addSubview:button];
    }


    {   // image维持原来的尺寸.
        UIButton *button = [UIButton buttonWithType:UIButtonTypeCustom];
        button.frame = CGRectMake(20, 200, 44, 44);
        [button setImage:[UIImage imageNamed:@"add"] forState:UIControlStateNormal];
        button.backgroundColor = [UIColor redColor];
        // left 让他不再遵从水平方向的居中。而是以button左边内边距28
        //  注意left有值就不再是居中后再加上left的距离。而是直接以left距离为依据
        //  top,bottom= 0 表示依然遵从垂直方向居中
        //   在image"内部"插入一个你“看不见”的内边距
        button.imageEdgeInsets = UIEdgeInsetsMake(0,28,0,0);
        [self.view addSubview:button];

        UIView *blueView = [[UIView alloc] init];
        blueView.backgroundColor = [UIColor blueColor];
        blueView.frame = CGRectMake( 0, 14, 28, 16);
        [button addSubview:blueView];
    }

}

@end
